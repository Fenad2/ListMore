# ListMore
提供一些小功能

## 功能列表

**复制准星目标 ID**

复制准星所指的方块或实体 ID

**实体高亮边框**

为名单中的实体类型添加原版发光描边效果

**实体高亮边框-选择列表**

每行一个实体 ID

**实体高亮边框-颜色**

设置名单中实体的描边颜色

**实体渲染黑名单开关**

跳过名单中实体的客户端渲染

**实体渲染黑名单-列表**

每行一个实体 ID

**实体渲染黑名单-范围**

黑名单实体在该三维范围内会重新渲染，0 为始终不渲染

**掏炉灰助手-开关**

高亮输入槽含有无法烧制物品的普通熔炉

**掏炉灰助手-范围设置**

以玩家所在区块为中心，区块为范围的扫描半径

**TNT爆炸预览**

以TNT当前实时位置预览方块破坏范围，红色为保守范围，黄色为额外可能范围

**TNT爆炸预览-显示模式**

选择是否显示可能被炸坏的方块

**玩家追踪线**

连线到附近玩家

**玩家追踪线颜色**

设置追踪线的颜色

**投射物落点预测**

显示支持投射物的预测落点
