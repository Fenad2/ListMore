# AimTrack Mod

Minecraft 1.21 Fabric 投射物落点预测与自瞄辅助。

## 功能

- **落点预测**：弓箭、三叉戟、雪球、鸡蛋、末影珍珠的飞行轨迹与命中点
- **自动瞄准**：弓箭满弓 / 三叉戟蓄力完成后自动锁定最近实体
- **碰撞箱高亮**：锁定目标显示红色碰撞箱

## 按键

| 按键 | 功能 |
|------|------|
| `R` | 切换自瞄开关 |

## 计算方式

**轨迹模拟**：逐 Tick 模拟投射物飞行，应用与原版一致的阻力与重力公式

```
vx *= drag
vy *= drag - gravity
vz *= drag
```

**性能优化**：
- 玩家状态未变化时直接读取缓存，避免每帧重复计算
- 实体搜索预缓存，模拟期间不再查询世界
- 速度低于阈值时提前终止模拟
- 数学运算使用原始 double，避免临时对象

**自瞄算法**：
1. 粗扫描：以2°步长遍历±20°范围，找到大致最优区间
2. 精修：在最优区间附近以0.2°步长精细搜索
3. 评分函数：通过完整弹道模拟，评估每个pitch角的命中偏差
4. 平滑转向模拟人手操作

## 支持投射物

| 投射物 | 落点预测 | 自瞄 |
|--------|---------|------|
| 弓箭 | 是 | 满弓时 |
| 三叉戟 | 是 | 蓄力时 |
| 雪球 | 是 | 否 |
| 鸡蛋 | 是 | 否 |
| 末影珍珠 | 是 | 否 |

## 安装

1. 安装 [Fabric Loader](https://fabricmc.net/use/)
2. 将 `aimtrack-mod-*.jar` 放入 `mods` 文件夹
3. 启动游戏

## 依赖

- Fabric API
- Minecraft 1.21
